# WorkflowFileBrowser
An Alfred workflow to browse workflow files
## Usage:
### Script
Usage:

    workflowsearch.py <query> [-c|-b|-r|-a|-w] [<keyword>]
    
Options:

    -c    Filter by category
    -b    Filter by bundleid
    -r    Filter by readme
    -a    Filter by createdby
    -w    Show all
